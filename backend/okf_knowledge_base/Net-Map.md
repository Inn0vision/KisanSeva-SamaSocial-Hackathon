# Net-Map
An empirical, participatory social networking tool produced by IFPRI to assess and improve complex governance systems. It is used for research, organizational development, and strategic network planning. Participants collaboratively sketch a network map of involved actors and stakeholders, mapping connections and using physical checkers pieces to build "influence towers" that visualize power dynamics and hierarchies. It has been used by organizations including IFAD, the World Bank, and WHO.
