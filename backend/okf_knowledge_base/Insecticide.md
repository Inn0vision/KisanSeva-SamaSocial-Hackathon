# Insecticide
Insecticides are chemical inputs used for pest control. Inappropriate use of insecticides can be counterproductive, potentially inducing [[Pests]] outbreaks by disrupting the natural population of predators that help control pests. Sustainable intensification advocates for carefully targeting the use of external inputs like insecticides.
