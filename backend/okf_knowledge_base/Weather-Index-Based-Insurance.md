# Weather-Index Based Insurance
A financial risk management tool that provides payouts to insured individuals based on a predetermined weather index (e.g., rainfall levels) rather than assessed losses. It is an innovative adaptation tool that can protect farmers from climate shocks. Safety nets can serve as a platform to improve access to this insurance for the poor, for example through "insurance-for-work" schemes, as demonstrated by the R4 Rural Resilience Initiative.
