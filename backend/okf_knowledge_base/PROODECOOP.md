# PROODECOOP
PROODECOOP is a coffee-growers' organization in Nicaragua that was part of the AdapCC initiative pilot. It developed an action plan to manage climate risks, which included training members on climate change, establishing a meteorological station, validating organic input fabrication, and implementing efficient water use techniques.
