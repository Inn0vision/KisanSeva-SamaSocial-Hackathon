# Quick Institutional Context Assessment
A simple methodology to help project developers build a useful picture of the institutional context for a Climate-Smart Agriculture (CSA) initiative. It involves five key steps: creating a general overview of relevant plans and institutions; mapping institutional interests and mandates; identifying institutional strengths; identifying institutional weaknesses; and determining the scope for synergies and collaborations between institutions.
