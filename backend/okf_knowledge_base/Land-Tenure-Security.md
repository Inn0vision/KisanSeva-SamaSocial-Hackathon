# Land Tenure Security
Secure land rights that provide the enabling environment for investments in sustainable land and water management, which are crucial elements of Climate-Smart Agriculture. Tenure security promotes and generates incentives for long-term investment in sustainable practices and is essential for establishing benefit distribution mechanisms for payments for ecosystem services.
