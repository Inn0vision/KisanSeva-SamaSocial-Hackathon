# Integrated multitrophic aquaculture (IMTA)
An aquaculture system that explicitly incorporates species from different trophic levels (e.g., fish, shellfish, seaweed) in the same system. Outputs (often waste nutrients) from one species become inputs (food) for another. This leads to greater efficiency, reduces environmental impact, and can improve the resilience of the farming system.
