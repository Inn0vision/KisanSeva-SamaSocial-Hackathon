# Extensive coastal pond/lagoons
A traditionally unfed aquaculture system for some warm-water species, using traps or nets. It may involve stocked fry and some molluscs. This system faces climate change issues such as slight changes in productivity, species temperature ranges, runoff, and salinity changes. Responses include changing harvesting patterns, increasing inputs/stocking, improving management, and possible expansion opportunities due to sea-level rise.
