# Hyogo Framework for Action
A global policy framework for Disaster Risk Reduction (DRR). It was adopted in 2005 and provided a comprehensive, action-oriented framework for reducing risks and building resilience to disasters. It is cited as a key part of the global enabling environment that supports DRR capacity, which in turn is harnessed to support Climate-Smart Agriculture (CSA).
