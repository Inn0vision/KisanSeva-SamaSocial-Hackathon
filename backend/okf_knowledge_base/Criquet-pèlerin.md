# Criquet pèlerin
The "Criquet pèlerin" (Desert Locust) is identified as a significant agricultural pest. The provided text references multilateral evaluations of campaigns against this pest from 2003-2005, as well as a mid-term evaluation report for the EMPRES program's "Criquet pèlerin" component in the Western Region. These evaluations highlight ongoing efforts to manage and control its outbreaks.
