# Resource Use Efficiency
A ratio between inputs (e.g., water, fertilizer) and outputs (agricultural production). It is a key component of sustainable agricultural intensification and Climate-Smart Agriculture (CSA). Increasing efficiency is a core adaptation strategy, particularly in areas where climate change is expected to reduce water availability, and is highly correlated with reduced emissions per unit produced. Investments to support increased efficiency include information generation (e.g., mapping water demand), research, extension, technology development (e.g., drip irrigation), and efficient input supply chains.
